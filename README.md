# Swoole4phalcon
Phalcon4 running on swoole4

**Requires**
- Swooole 4.x Extension
- Phalcon 4.x Extension
- PHP7.4

**Description**
- support $this->request->getRawBody() and $this->request->getJsonRawBody()

**Start**
```
composer install
php run serve
```
